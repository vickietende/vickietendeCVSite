Mecca Inventory Managent System Project Details

Mecca Private Limited is a Toyota Zimbabwe parts Agent and their busisnes is to import parts on behalf of Toyota Zimbabwe. Mecca Inventory Managent System is a Web Application built on the .Net Platform using C#, ASP.NET, Bootstrap 4, MSSQL Server, JavaScript, JQuery, and AJAX Tools. Its functions include Sales, Purchases, Expenses, Accounting, Inventory Management and several financial reports. Currently the project is almost complete and ready to be deployed on the live server by Jan 2022.
